# Дыхание · гимнастика Стрельниковой

Метроном-счётчик для дыхательной гимнастики Стрельниковой. Статический сайт (PWA):
открывается в браузере, ставится на главный экран iPhone как приложение, работает офлайн.

Живая версия: <https://boundlessend.github.io/strelnikova/>

## Возможности

- комплекс из 12 упражнений, состав круга настраивается чекбоксами
- счёт вдохов с крупным дисплеем и кольцом-прогрессом (по умолчанию 32 счёта)
- настраиваемый темп метронома (30-150 вдохов в минуту)
- число кругов (1-10), паузы между упражнениями и между кругами с обратным отсчётом
- звук метронома с акцентом на первом счёте, опционально голосовой счёт
- светлая и тёмная темы, зум страницы отключён
- тактильная отдача на Android и iOS 17.4-26.4 (в iOS 26.5 Apple закрыла вибро из веба)
- экран не гаснет во время упражнения, настройки сохраняются локально
- настройки фиксируются на старте тренировки; пауза и продолжение в любой момент

## Запуск локально

```bash
python3 -m http.server 8137
```

Открыть `http://localhost:8137`.

## Проверка

Smoke-тест Playwright прогоняет старт, паузу, продолжение, сброс
и полное завершение тренировки:

```bash
npm install
npx playwright install chromium
npm test
```

## Установка на iPhone

Открыть сайт в Safari, нажать «Поделиться» → «На экран Домой».
Дальше запускается на весь экран как обычное приложение, интернет не нужен.

## Стек

Чистые HTML, CSS и JavaScript без зависимостей и сборки.
Офлайн-работа через service worker, установка через web app manifest.
Playwright используется только как dev-зависимость для smoke-теста.
